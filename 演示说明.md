# AI字幕生成翻译系统 - 使用演示

## 🎉 项目完成！

我已经为你创建了一个完整的AI字幕生成和翻译系统，包含以下功能：

### ✨ 主要功能
- **🎤 音频转录**：使用OpenAI Whisper模型进行高精度语音识别
- **🌐 多语言翻译**：支持中文、英文、日文、韩文等多种语言翻译
- **📁 多格式支持**：支持MP4、MP3、WAV等常见音视频格式
- **💾 多种输出格式**：支持SRT、VTT、JSON等字幕格式导出
- **🌐 Web界面**：提供友好的用户界面
- **⚡ 命令行工具**：支持批量处理音频文件

### 🚀 快速开始

#### 1. Web界面使用
应用已经在 http://127.0.0.1:5000 启动，你可以：

1. **上传文件**：拖拽或点击上传音频/视频文件
2. **选择语言**：选择音频语言和目标翻译语言
3. **开始转录**：点击开始转录按钮，等待处理完成
4. **字幕翻译**：选择目标语言进行翻译（可选）
5. **下载字幕**：预览并下载生成的字幕文件

#### 2. 命令行工具使用
```bash
# 处理单个文件
python cli.py video.mp4 -t zh-cn

# 批量处理目录
python cli.py ./videos/ --recursive -t zh-cn

# 指定输出格式和模型
python cli.py audio.mp3 -f vtt -m large -t en
```

### 🔧 注意事项
1. **FFmpeg**：系统提示FFmpeg未安装，这会影响视频处理功能
   - 你可以从 https://ffmpeg.org/download.html 下载安装
   - 安装后添加到系统PATH环境变量

2. **首次使用**：首次运行时会下载Whisper模型，可能需要一些时间

3. **文件大小**：支持最大500MB的文件上传

### 🎯 使用场景
